# ecommerce-myshop
MyShop is a modern and user-friendly e-commerce web application designed to provide a seamless shopping experience for users.
It allows customers to browse products, add items to their cart, and complete purchases using multiple payment methods such as Cash on Delivery or Online Payment. The platform also includes basic admin functionality for managing products and orders.

User Registration & Login
Secure authentication system for users and admins.

Shopping Cart System
Add, update, and remove items before checkout.

Flexible Payment Options
Supports "Cash on Delivery" and simulated "Online Payment".

Order Management
Orders are recorded with detailed line items and payment tracking.

PDF Bill Generation
Customers can download their order bill in PDF format.

Admin Dashboard
Manage product listings, view orders, and monitor users.

Product Images
Supports image uploads and product image previews.

Tech Stack Used:
Frontend: HTML, CSS (Bootstrap), JavaScript

Backend: PHP

Database: MySQL

Libraries: FPDF (for PDF generation)

Server: XAMPP/Apache (Localhost)
